Fitted — Install Guide

Download started.

Follow these 4 steps to finish installing. Takes about 2 minutes.

1

Unzip the file

Find fitted-extension.zip in your downloads. Right-click → Extract All (Windows) or double-click (Mac). You'll get a folder called fitted-extension.

Keep this folder somewhere permanent — Desktop or Documents. Chrome needs it to run the extension.
2

Open Chrome Extensions page

Paste this in your address bar and hit Enter:

chrome://extensions

Or: Chrome menu (⋮) → Extensions → Manage Extensions
3

Turn on Developer Mode

Find the "Developer mode" toggle in the top-right corner. Switch it on.

This is safe and standard for extensions not yet on the Chrome Web Store.
4

Load the extension

Click "Load unpacked" in the top-left. Navigate to the fitted-extension folder you unzipped. Select it.

The Fitted icon should now appear in your Chrome toolbar.

You're ready.

Open any job listing on LinkedIn, Indeed, or Naukri. Fitted will auto-detect the JD and show your match analysis in the side panel.

It'll ask you to upload your resume the first time — after that, it works automatically on every job page you visit.
